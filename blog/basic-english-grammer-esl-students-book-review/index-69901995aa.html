I read the book, “Basic English Grammar for ESL Students“ for learning English
grammar again.

Some advanced points in the book are as follows:

  • It’s so clear to understand grammar because the instructions are easy to read for non-native English speakers at the beginner level.
  • The grammar is quite simple, so anyone can understand it.
  • There are a lot practice exercises.
  • The price of the book is only $2.71, thusly we can try it easily!

🚌 Impressive points

Some points which I was impressed by in the book are as follows:

🐞 PRESENT SIMPLE

  • TO BE => It’s used in sentences about name/country/age/emotions/jobs.
  • VERB => It happens regularly or it is generally true.

🎳 A/AN, THE

  • A/AN => general (one of many), THE => specific(one specific)

🍣 THIS/THESE, THAT/THOSE

  • THIS/THESE => near, THAT/THOSE => far

🤔 AUXILIARY VERB (CAN, MUST, HAVE TO)

  • CAN => To be able to
  • MUST/HAVE TO => It’s necessary/obligatory

👽 IN, ON, AT

  • IN

    • TIME => Months, Seasons, Years, Periods of the day
    • PLACE => Cities, Countries, Rooms, Buildings, Closed spaces

  • ON

    • TIME => Date and days
    • PLACE => Transportation, Surfaces

  • AT

    • TIME => Times
    • PLACE => Contexts, Events, Buildings, etc.

😸 UNCOUNTABLE NOUNS

  • LIQUIDS AND SOME FOODS => water, butter, rice
  • IDEAS AND CONCEPTS => love, fun, work, money
  • INFORMATION => advice, knowledge, news
  • CATEGORIES => music, furniture, equipment, jewelry

🎉 GOING TO

Use “going to “ to talk about future events, plans, and predictions.

🗻 PRESENT PERFECT

  • Experience in the past, when we don’t know or don’t say exactly when the experience happened
    • e.g) I have been to Italy.
  • An action or state that started in the past and continues to the present
    • e.g) I have been married for a long time.
